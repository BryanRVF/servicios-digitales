Términos y Condiciones

Última actualización: Enero 2024

1. Servicio

DataValidator API proporciona servicios de validación de datos (emails, teléfonos, documentos) mediante API REST.

2. Uso del servicio

El cliente se compromete a usar el servicio de forma legal y ética. No está permitido usar el servicio para actividades fraudulentas o ilegales.

3. Facturación y pagos

Los pagos se procesan mensualmente según el plan contratado. La facturación comienza después del periodo de prueba gratuito.

4. Limitación de responsabilidad

El servicio se proporciona "tal cual" sin garantías de ningún tipo. DataValidator API no garantiza la precisión al 100% de las validaciones.

5. Cancelación

Puedes cancelar tu suscripción en cualquier momento desde el dashboard.

6. Contacto

Para preguntas sobre estos términos: contacto@datavalidatorapi.com
